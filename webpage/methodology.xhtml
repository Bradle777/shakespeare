The Markup

Meta Elements

Element Order

  • 'created' - recorded year, if applicable
  • 'main_title' - name of play
  • 'author' - William Shakespeare
  • 'toc' - includes a list of the act and scene breakdown, if applicable; only for plays
  • 'character_list' - lists all characters; main characters are given chara elements to define them further
  • 'chara' - an element wrapped around character name and description, include name and gender attribute
  • 'setting' - overall location of the play

Body Elements

Presentational Elements

  • 'act' - wrapped around full act, number attribute 1 to 5
  • 'scene' - wrapped around each scene, numbered so as not to repeat for a new act (1-14)
  • 'title' - after the act and scene element, include Act and Scene titles as well as the setting
  • 'sp' - short for "speaker," wrapped around each character who speaks with 'who' and 'gender' attributes for main characters
  • 'stage' - for stage directions, in brackets -- including: Exit, Exeunt, Enter, aside, other commands

Descriptive elements

'Tone' element

  • Contains larger portions of text that represents a characters emotional state

Attributes and values

vibe
"anger" - speech with hints of frustration
"sad" - speeches of grieving/mourning
"guilt" - speech of regret
"romantic" - speech with phrases of love
"happy" - speech of celebration
"begging" - speech of weakness, a negative power dynamic

'Depiction' element

  • Contains phrases or words that portray a character in a specific way

Attributes and values

role
"wife" - role of women as a care taker or object of husband
"mother" - role of women as a care taker
"villain" - evil actions or phrases
"victim" - attacked by villain
"smart" - shown to have knowledge
"insanity" - character being called crazy
"race" - depictions of skin color
"weakness" - words like tears/weep, indicators of vulnerability
"manipulation" - phrases or sections where a character lies to control another
"impure" - references to physical characteristics that make someone not pure
"respect" - phrase indicating a positive power dynamic
"pure" - painting character as kind, pure

'Power' element

  • any depictions of the control/respect of one person
  • gender depictions in this context also mentioned with attribute

Attributes and values

role
"patriarch" - words like lord, prince, highness to depict institutional male power
"matriarch" - words like queen, majesty, princess to depict a woman in a position of control
"aside" - stage direction aside is given as a way to speak to the audience, denotes a power Shakespeare gave that character

'Color' element

  • any mentions of a color, and what that color means in context of the speech or what is represents

Attributes and values

'is' - specifying what the color is
"black"
"white"
"red"
"gold"
'context' - explaining what the color is used to describe
"impure"
"pure"
"death"
"savage"
"money"
"grief"
"appearance"
"race"

'Violence' element

  • sentence or phrase that contains verbal or physical attacks
  • gender depictions in this context also mentioned with attribute

Attributes and values

type
"threat" - proposed violence, promises of future harm
"insult" - verbal attack
"murder" - line(s) where a character is killed
"plotting" - character(s) planning violence
"sex" - sexual violence, rape, inappropriate lust

'Love' element

  • A sentence or phrase that contains moments of sincere love of another character

Attributes and values

action
"wedding" - references to marriage or a marriage ceremony
"verbal" - spoken claims of love without physical action
"physical" - love that involves physical contact

'gender' attribute

  • can be applied to almost any element to specify the gender, if important gendered difference

Values

"female"
"male"
"???" - unspecified or unsure of gender
